## 修饰符

1. 事件修饰符

- once 只执行一次

```vue
<button @click.once="handleClick">点击</button>
```

- self 只有点击当前元素时才会触发

```vue
<button @click.self="handleClick">点击</button>
```

- prevent 阻止默认行为

```vue
<a href="https://www.baidu.com" @click.prevent="handleClick">点击</a>
```

- stop 阻止事件冒泡

```vue
<template>
  <div @click.stop="handleClick">
    <button @click.stop="handleClick">点击</button>
  </div>
</template>
```

- capture 使用事件捕获模式

```vue
<button @click.capture="handleClick">点击</button>
```

- passive 事件默认行为不会被阻止

```vue
<!-- 告诉浏览器不阻止默认滚动行为 -->
<div @scroll.passive="handleScroll"></div>
```

2. 按键修饰符

- enter 回车键

```vue
<input type="text" @keyup.enter="handleClick" />
```

- delete 删除键

```vue
<input type="text" @keyup.delete="handleClick" />
```

- esc 退出键

```vue
<input type="text" @keyup.esc="handleClick" />
```

- space 空格键

```vue
<input type="text" @keyup.space="handleClick" />
```

- tab tab 键

```vue
<!-- 阻止 tab 键默认行为-->
<input type="text" @keyup.tab.prevent="handleClick" />
```

- up 上键

```vue
<input type="text" @keyup.up="handleClick" />
```

- down 下键

```vue
<input type="text" @keyup.down="handleClick" />
```

- left 鼠标左键

```vue
<input type="text" @click.left="handleClick" />
```

- right 鼠标右键

```vue
<input type="text" @click.right="handleClick" />
```

- middle 鼠标中键

```vue
<input type="text" @click.middle="handleClick" />
```

3. 表单输入修饰符

- .lazy 只在 change 事件中同步

```vue
<input type="text" v-model.lazy="name" />
```

- .number 自动将输入值转为数值类型

```vue
<input type="text" v-model.number="age" />
```

- .trim 自动过滤用户输入的首尾空白字符

```vue
<input type="text" v-model.trim="name" />
```
